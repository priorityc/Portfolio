Food Web App

The Food Web Application was built using React.js for its modular architecture and dynamic UI capabilities, paired with CSS Modules for scoped styling and maintainability. Real-time recipe data is sourced from the Spoonacular API, with dietary filters and nutrition info integrated seamlessly.

Users browse or search for meals based on dietary preferences, ingredients, or cooking time.Selected recipes are added to a weekly planner, with drag-and-drop functionality and calendar integration. Shoping list generation-The app compiles ingredients from selected meals into a categorized shopping list, optimized for mobile use that user can easy save.
