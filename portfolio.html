Yorvik Parking App

A mobile-first parking solution Prototype built with React Native.

Designed an intuitive prototype for real-time longboat parking space tracking using React Native and Express.js.

Designed initialy with Vite tooling the prototype has been adjusted to use Expo Go whilist remain that the application communicates with custom-build Express.js backend to simulate the booking marine experience. Captain can register check availible berths for specific time periods and generate permit confirmation with QR code. The intuitive interface ensures fluid interaction flow. The main purpose of the prototype is to showcase the features without needing the live beckend -perfect for organisations or buissiness that require visual representation of their service or exploration of technology tools for integrating services.

Key Features

Technologies used:

React Native, Express.js, Vite, OBS video

  • Real-time Parking Availability
  • Location-Based Search
  • Secure Booking System

Cloud Architecture Design – YPC Integration Project

As part of my cloud architecture training, I designed a scalable hybrid solution using AWS and OpenStack to unify YPC’s mobile and web-based systems. The system supports authentication, payments, photo storage, and admin/frontend hosting.

Overview

The YPC parking management system needed a unified cloud-based architecture to integrate its web and mobile applications. My solution ensured seamless service expansion while supporting key functionalities such as user accounts, payment handling, and API-driven interactions.

Challenges

  • Existing applications were independent, requiring a shared backend.
  • Secure cloud-based payment processing with scalability.
  • Optimized storage for photos and user data.
  • Ensuring future service expansion without disrupting current operations.

System integration architecture using AWS services, OpenStack, and a centralized backend.

Food Web application -platform

Recipe exploration platform that allow users to explore receipts ideas.

Overview

Food App is a recipe exploration platform that allows users to: - Search for recipes using an API (e.g., Spoonacular) - View detailed ingredients and cooking steps - Save favorite recipes for easy access - Filter recipes based on dietary preferences .

How it works

Uses fetch to retrieve data from the Spoonacular API.

React manages UI state (e.g., selected recipes, search history).

Challenges

  • API request limits causing slow searches
  • Implemented debouncing with useEffect to optimize requests
  • Ensuring seamless UI interaction
  • Used React hooks & conditional rendering to improve responsiveness
