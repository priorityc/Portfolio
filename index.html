Food Web application -platform

Recipe exploration platform that allow users to explore receipts ideas.

Overview

FoodApp is a calorie-conscious meal planner designed to reduce weekly food stress by offering personalized recipes and a downloadable shopping list—making nutrition planning easier, faster, and more sustainable.

How it works

Uses fetch to retrieve data from the Spoonacular API.

React manages UI state (e.g., selected recipes, search history, weekly meal plan).

Challenges

  • Planning Overwhelm: Users often struggle to organize diverse, balanced meals across the week. FoodApp auto-generates structured plans for breakfast, lunch, and dinner using real-time recipe data.
  • Nutritional Awareness:It brings transparency to calorie intake, helping users manage fitness goals or dietary restrictions while avoiding guesswork.
  • Ingredient Shopping Efficiency: By aggregating ingredients across the week and removing duplicates, it creates a clean, downloadable shopping list—saving time and reducing food waste.
  • Responsive, Accessible UI: The app offers a responsive, semantic layout that adapts across devices while ensuring meals are clearly labeled, organized, and accessible.

Cosmic Black Holes event-CSS animations/Landing page

Landing page animation

Overview

A visually immersive landing page designed to highlight the expressive power of pure CSS animations. This project blends minimalist layout with dynamic motion to create an atmospheric experience that feels both modern and playful.

How it works

The page uses keyframe animations, transform effects, and opacity transitions to animate elements like headings, icons, and backgrounds.

Subtle motion guides the user’s attention while maintaining a clean, uncluttered interface.

Goal: Create a CSS-based animation simulating gravitational lensing.

Challenges: Achieved distortion effects using gradients and keyframes. Ensured performance and accessibility.

Outcome: Demonstrated creative use of CSS to convey complex astrophysical concepts.
